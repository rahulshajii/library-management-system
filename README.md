# 📚 Library Management System

A full-stack Library Management System built with **Angular**, **Django REST Framework**, and **SQLite**.

## 🚀 Features

* Dashboard with live statistics
* Book Management
* Member Management
* Transaction History
* Issue Book
* Return Book
* Fine Calculation
* REST API Integration
* Responsive Bootstrap UI

## 🛠️ Tech Stack

### Frontend

* Angular 19
* TypeScript
* Bootstrap 5
* Bootstrap Icons

### Backend

* Django
* Django REST Framework
* SQLite

## 📂 Project Structure

```
library-management-system/
│
├── backend/
│   ├── books/
│   ├── members/
│   ├── transactions/
│   └── config/
│
└── frontend/
    ├── src/
    └── app/
```

## ▶️ Run the Backend

```bash
cd backend
source venv/bin/activate
python manage.py runserver
```

Backend runs at:

```
http://127.0.0.1:8000
```

## ▶️ Run the Frontend

```bash
cd frontend
ng serve
```

Frontend runs at:

```
http://localhost:4200
```

## 📡 API Endpoints

* `/api/dashboard/`
* `/api/books/`
* `/api/members/`
* `/api/transactions/`
